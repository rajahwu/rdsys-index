# **4\. Iconography**

Radiant Systems icons are infrastructure signage:

* 2px stroke  
* No rounded corners  
* Geometric, strict  
* Zero personality  
* Designed for clarity, not charm

### **Sizes**

* 16×16px — inline UI  
* 24×24px — navigation  
* 32×32px — system diagrams

### **Categories**

* Governance  
* Logs  
* Alerts  
* Infrastructure  
* System Maps  
* Permissions

### **Color Rules**

* Default: text.secondary  
* Active: brand.teal.500  
* Disabled: foundation.gray.300
