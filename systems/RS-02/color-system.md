# **2\. Color System**

Radiant Systems uses a muted, institutional palette designed for clarity, legibility, and long-term stability. No bright colors, no gradients, no decorative saturation.

### **Foundation Colors**

| Token Name | Hex | RGB | CMYK | Usage |
| ----- | ----- | ----- | ----- | ----- |
| foundation.white | \#FFFFFF | 255,255,255 | 0,0,0,0 | Pure white surfaces |
| foundation.cream.50 | \#FCFCF9 | 252,252,249 | 1,1,2,0 | Page backgrounds |
| foundation.cream.100 | \#FFFFFD | 255,255,253 | 0,0,1,0 | Cards, panels |
| foundation.gray.200 | \#F5F5F5 | 245,245,245 | 4,3,3,0 | Dividers, borders |
| foundation.gray.300 | \#A7A9A9 | 167,169,169 | 38,30,31,0 | Disabled states |

### **Color Behavior**

* Hover states use 5–10% teal overlays  
* Pressed states use 20% teal overlays  
* Never use pure black  
* Never use saturated colors  
* Never use gradients
