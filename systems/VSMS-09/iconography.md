# **4\. Iconography**

Style: Clean outlined icons with 2px stroke weight. Rounded corners (2px) for approachability. Geometric precision balanced with organic warmth — not too technical, not too casual.

Stroke Width: 2px (consistent across all icons)  
Corner Radius: 2px on internal corners, rounded stroke caps  
Sizes: 20×20px (compact UI), 24×24px (standard), 32×32px (features)  
Default Color: neutral.mid (\#9E9E9E)  
Active Color: brand.primary (\#1C2C4B)  
Success Color: accent.success (\#88A57B)

Icon Categories:

Phase Icons (VSM Methodology):

* Genesis (seed sprouting)  
* Source Code (foundation blocks)  
* Powerhouse (energy core)  
* Excellence (refined gem)  
* Mastery (summit peak)

Tool/Action Icons:

* Practice (repetition symbol)  
* Reflect (thought bubble)  
* Progress (upward arrow)  
* Complete (checkmark)  
* Bookmark (save for later)

Navigation Icons:

* Home (house outline)  
* Lessons (stacked pages)  
* Progress (ascending chart)  
* Resources (toolbox)  
* Profile (user circle)

Status Icons:

* In Progress (circular progress)  
* Completed (filled circle with check)  
* Locked (padlock)  
* Available (open door)

Icon Grid:  
All icons designed on 24×24px base canvas with 20×20px live area (2px padding). Align to 2px grid for pixel-perfect rendering.

Special Glyph:  
VSM Peak — Stylized mountain peak forming a "V" shape. This is the primary brand mark, appears on logo lockup, app icon, achievement badges, phase completion markers.
