Type Scale:

* Display: 36px / 1.2 — Marketing hero, landing pages  
* H1: 30px / 1.25 — Page titles  
* H2: 24px / 1.3 — Section headers  
* H3: 20px / 1.4 — Card titles  
* H4: 18px / 1.4 — Minor headings  
* Body Large: 18px / 1.75 — Lead content  
* Body: 16px / 1.5 — Standard content  
* Body Small: 14px / 1.5 — Secondary content  
* Caption: 12px / 1.5 — Micro text  
* Mono: 14px / 1.6 — Code/logs

Typography Notes:  
Space Grotesk provides geometric precision with personality — perfect for a systematic pipeline brand. Inter ensures clarity for UI and content. JetBrains Mono essential for code snippets, logs, and technical displays (developer-centric). Slightly tighter letter-spacing on display text for optical balance.

Fallback Stacks:

* Space Grotesk: 'Space Grotesk', 'Archivo', 'Work Sans', sans-serif  
* Inter: 'Inter', \-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif  
* JetBrains Mono: 'JetBrains Mono', 'Fira Code', 'SF Mono', Consolas, monospace
