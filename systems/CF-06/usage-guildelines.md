# **6\. Usage Guidelines**

Spacing:

* Base unit: 4px (0.25rem)  
* Spacing scale: xs (4px), sm (8px), md (16px), lg (24px), xl (32px), 2xl (48px), 3xl (64px)  
* UI spacing: 8px between related elements, 16px between groups, 24px between sections  
* Card padding: 16px (compact), 24px (standard), 32px (spacious)  
* Container padding: 16px (mobile), 24px (tablet), 32px (desktop)

Corner Radius:

* Small: 4px (buttons, badges, tags)  
* Medium: 8px (cards, inputs, standard surfaces)  
* Large: 16px (modals, panels, feature containers)  
* Full: 9999px (circular elements, pills)

Shadows:

* Subtle (sm):  
  box-shadow: 0 1px 2px rgba(0, 0, 0, 0.05);  
  Usage: Slight elevation for cards  
* Standard (md):  
  box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);  
  Usage: Hover states, dropdowns  
* Prominent (lg):  
  box-shadow: 0 10px 15px rgba(0, 0, 0, 0.15);  
  Usage: Modals, important panels  
* Extra Prominent (xl):  
  box-shadow: 0 20px 25px rgba(0, 0, 0, 0.2);  
  Usage: Major overlays, tooltips

Borders:

* Thin: 1px — standard borders, dividers  
* Thick: 2px — emphasis borders, active states

Photography & Imagery:

* Usage: Rare — prefer diagrams and dashboard UI  
* When used: Screenshots of actual pipelines in action, before/after comparisons  
* Style: Clean, modern, well-lit workspace shots if needed  
* Avoid: Stock photos, decorative imagery, anything that doesn't show actual process

Logo Usage:

* Primary lockup: Factor X mark \+ "CONTENT FACTOR" wordmark (horizontal)  
* Icon-only: X mark in circle for app icons, favicons  
* Minimum size: 140px width for lockup, 32px for icon  
* Clear space: Maintain padding equal to height of X mark on all sides  
* Backgrounds: Works on neutral.background, white, or dark surfaces (use light version)  
* Prohibited uses:  
  * Don't stretch or distort the X geometry  
  * Don't rotate or tilt  
  * Don't apply gradients or effects  
  * Don't use colors outside pipeline palette  
  * Don't place on busy backgrounds  
  * Don't crowd with other elements inside clear space
