# **4\. Iconography**

Style: Thin-line, geometric icons with consistent stroke weight. More refined and technical than street-style — think UI precision meets motion design. Clean outlines with subtle rounded corners.

Stroke Width: 2px (consistent across all icons)  
Corner Radius: 2px on internal corners, rounded stroke caps  
Sizes: 16×16px (inline), 20×20px (compact), 24×24px (standard), 32×32px (features)  
Default Color: neutral.mid (\#6B7280)  
Active Color: brand.primary (\#006E6D)  
CTA Color: brand.secondary (\#C41230)

Icon Categories:

Brand/Motion Icons:

* Frame (brackets or crop marks)  
* Motion timeline (animation frames)  
* Grid system (modular layout)  
* Play button (motion/launch)  
* Layers (depth/stacking)

Service Icons:

* Brand identity (shield/badge)  
* Systems architecture (connected nodes)  
* Creative production (palette/brush)  
* Ecommerce (shopping cart/storefront)  
* Launch campaigns (rocket/megaphone)

Navigation Icons:

* Home (house outline)  
* Services (grid)  
* Portfolio (folder/gallery)  
* About (info circle)  
* Contact (envelope)

Status Icons:

* Active (filled circle with pulse)  
* Processing (loader/spinner)  
* Complete (checkmark circle)  
* Alert (warning triangle)

Icon Grid:  
All icons designed on 24×24px base canvas with 20×20px live area (2px padding). Align to 2px grid for pixel-perfect rendering at standard sizes.

Special Glyph:  
Frame Motif — Stylized brackets or crop marks forming a frame. This is Dropframe's signature mark, representing:

* Film frames dropping in sequence  
* UI grids snapping into place  
* Modular systems aligning perfectly

Used in:

* Logo lockup (integrated with wordmark)  
* App icon/favicon (DF monogram inside frame)  
* Section dividers  
* Loading states
