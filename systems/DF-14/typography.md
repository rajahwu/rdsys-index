Type Scale:

* Display: 40px / 1.2 — Marketing hero, landing pages  
* H1: 32px / 1.25 — Page titles  
* H2: 24px / 1.3 — Section headers  
* H3: 20px / 1.4 — Subsections, cards  
* H4: 18px / 1.4 — Minor headings  
* Body Large: 18px / 1.6 — Lead content  
* Body: 16px / 1.6 — Standard content  
* Body Small: 14px / 1.6 — Secondary content  
* Caption: 12px / 1.5 — Micro text

Typography Notes:  
Inter provides crisp, modular, tech-forward clarity — perfect for a design-systems brand. Libre Franklin ensures clean, accessible readability for body content. Display text uses tighter letter-spacing for optical balance. Buttons use uppercase with looser tracking (+0.05em) for clarity and impact. Line heights are generous (1.6 for body) to support focused reading.

Fallback Stacks:

* Inter: 'Inter', \-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif  
* Libre Franklin: 'Libre Franklin', 'Work Sans', 'Open Sans', sans-serif

Alternative Typography Direction: If you prefer a more geometric feel, consider Manrope for headings. It maintains the same weights and can swap in directly.
