### **4\. Iconography**

Style: Flat, minimal, 2px stroke or solid fill. Geometric precision with rounded corners for approachability. Consistent padding and optical alignment across all sizes.

Stroke Width: 2px (scales proportionally: 1.5px at 16px, 2px at 24px, 3px at 32px)  
Corner Radius: 2px on internal corners, fully rounded caps on stroke ends  
Sizes: 16×16px (inline UI), 24×24px (navigation, standard), 32×32px (features), scalable SVG  
Default Color: neutral.500 (\#6B7280)  
Active Color: brand.primary (\#3B82F6)  
Disabled Color: neutral.300 (\#D1D5DB)

Icon Categories:

* Token Icons: Color swatch, Typography "A", Spacing ruler, Shadow layers, Grid template  
* Nav Icons: Tokens, Components, Docs, Search, Settings, Home  
* Status Icons: Success checkmark, Warning triangle, Error X, Info circle, Loading spinner  
* Action Icons: Edit pencil, Delete trash, Copy duplicate, Download arrow, Upload arrow, Plus add, Minus remove  
* UI Icons: Chevron (up/down/left/right), Menu hamburger, Close X, Expand arrows, Collapse, External link

Icon Grid:  
All icons designed on 24×24px base canvas with 20×20px live area (2px padding on all sides). Align to 2px grid for pixel-perfect rendering at standard sizes.

Special Considerations:  
Style System icons prioritize clarity and immediate recognition over decoration. Stroke weight remains consistent across complexity — simple icons don't get thinner strokes. All icons tested at 16px for legibility.
