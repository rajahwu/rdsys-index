Type Scale:

* Display: 48px / 1.1 — Marketing hero text, landing pages  
* H1: 32px / 1.25 — Page titles, primary headings  
* H2: 24px / 1.3 — Section headers  
* H3: 20px / 1.4 — Subsections, cards  
* H4: 18px / 1.4 — Minor headings  
* Body Large: 18px / 1.6 — Lead paragraphs  
* Body: 16px / 1.5 — Standard content  
* Body Small: 14px / 1.5 — Secondary info  
* Caption: 12px / 1.5 — Micro text  
* Mono: 14px / 1.6 — Code/tokens

Typography Notes:  
Inter provides exceptional clarity across all sizes with excellent web rendering. Bold (700) is reserved for major headings only; SemiBold (600) for subsections; Medium (500) for buttons. Display text uses tighter letter-spacing for optical balance. JetBrains Mono essential for technical content, token display, code examples.

Fallback Stacks:

* Inter: Inter, \-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Helvetica, Arial, sans-serif  
* Mono: 'JetBrains Mono', 'Fira Code', 'SF Mono', Monaco, Consolas, monospace
