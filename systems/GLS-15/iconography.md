# **4\. Iconography**

Style: Bold, solid-fill icons with strong presence. More weight than typical line icons — think street signage and industrial marks. Geometric but organic, imperfect edges welcome.

Stroke Width: 2.5px (thicker than refined systems — more presence)  
Fill Style: Solid fills preferred over outlines for key icons  
Corner Radius: 2-3px, slightly more rounded than sharp geometric  
Sizes: 20×20px (compact), 24×24px (standard), 32×32px (features), 40×40px (hero)  
Default Color: neutral.mid (\#6B7280)  
Active Color: grindline.teal (\#00A6A6)  
CTA Color: grindline.coral (\#FF5A5F)  
Success Color: grindline.gold (\#FFC857)

Icon Categories:

Service Icons:

* Brand Systems (modular blocks, construction)  
* Creative Production (paint palette, spray can)  
* Ecommerce (shopping bag, storefront)  
* Launch Campaigns (megaphone, rocket)  
* Ops Consulting (clipboard, checklist)

Action Icons:

* Build (hammer, wrench)  
* Create (pencil, brush)  
* Launch (rocket, arrow up)  
* Connect (link, handshake)  
* Progress (upward arrow, growth)

Navigation Icons:

* Services (grid blocks)  
* Work (briefcase, toolbox)  
* Team (people, community)  
* Contact (speech bubble)  
* Menu (stacked lines)

Status Icons:

* Active (filled circle with pulse)  
* Grinding (rotating gear)  
* Complete (solid checkmark)  
* Queued (stacked dots)

Icon Grid:  
All icons designed on 24×24px base canvas with 20×20px live area (2px padding). Less strict pixel-grid alignment than Dropframe — slight organic variation acceptable.

Special Glyph:  
Grindline Mark — Stylized "G" or abstract grinding wheel/gear shape. Represents continuous motion, momentum, and the grind itself. Used in:

* Logo lockup (icon mark \+ wordmark)  
* Favicon/app icon  
* Loading states (animated rotation)  
* Corner stamps on work samples  
* Achievement badges
