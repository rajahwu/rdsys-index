# **6\. Usage Guidelines**

Spacing:

* Base unit: 8px — foundation for all spacing  
* Spacing scale: 8px, 16px, 24px, 32px, 48px, 64px  
* UI spacing: 16px between related elements, 24px between groups, 32px between major sections  
* Card padding: 20px (compact, tighter than Dropframe), 32px (spacious)  
* Container padding: 16px (mobile), 24px (tablet), 32px (desktop)

Corner Radius:

* Small: 4px (buttons, tags)  
* Medium: 6px (cards, inputs — slightly softer than geometric 8px)  
* Large: 12px (feature sections, modals)  
* Organic: Variable 8-16px for hand-drawn elements

Shadows:

* Subtle (sm):  
  box-shadow: 0 2px 4px rgba(46, 46, 46, 0.15);  
  Usage: Card lift, subtle depth  
* Standard (md):  
  box-shadow: 0 4px 8px rgba(46, 46, 46, 0.2);  
  Usage: Hover states, dropdowns  
* Heavy (lg):  
  box-shadow: 0 8px 16px rgba(46, 46, 46, 0.3);  
  Usage: Modals, major overlays — stronger than Dropframe  
* Glow (teal):  
  box-shadow: 0 0 20px rgba(0, 166, 166, 0.4);  
  Usage: Active states, neon-sign effect  
* Focus ring:  
  box-shadow: 0 0 0 3px rgba(0, 166, 166, 0.5);  
  Usage: Keyboard focus (electric teal)

Borders:

* Thin: 1px — standard dividers  
* Medium: 2px — emphasis, active states  
* Heavy: 3-4px — strong emphasis, street-sign style

Textures:

* Grain overlay: Subtle noise texture at 5-10% opacity on dark backgrounds  
* Halftone: Dot patterns for depth without solid fills  
* Paper texture: Subtle paper grain on light backgrounds

Logo Usage:

* Primary lockup: Horizontal — Grindline mark \+ "GRINDLINE STUDIO" wordmark  
* Stacked: Mark above wordmark for square spaces  
* Icon-only: Grindline mark alone for app icons, favicons  
* Minimum size: 160px width for horizontal, 80px for stacked, 32px for icon  
* Clear space: Maintain padding equal to height of mark on all sides  
* Backgrounds: Works on grindline.dark, neutral.light, textured walls, urban photography  
* Prohibited uses:  
  * Don't stretch or distort proportions  
  * Don't rotate (unless intentionally "pasted" on angle for effect)  
  * Don't use colors outside palette  
  * Don't place on low-contrast busy backgrounds  
  * Don't sanitize the mark — embrace the grit
